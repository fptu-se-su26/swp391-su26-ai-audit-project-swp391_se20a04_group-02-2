# Hướng dẫn thêm hình ảnh sản phẩm

## Cách thêm ảnh vào project:

1. **Download ảnh** cho các sản phẩm sau (600x400px):
   - `thanh-long.jpg` - Thanh long ruột đỏ
   - `ca-phe.jpg` - Cà phê robusta
   - `gao-st25.jpg` - Gạo ST25
   - `ot-chuong.jpg` - Ớt chuông
   - `xoai-cat.jpg` - Xoài cát Hòa Lộc  
   - `hat-tieu.jpg` - Hạt tiêu

2. **Copy ảnh** vào folder này: `public/images/products/`

3. Sau khi copy xong, ảnh sẽ tự động hiển thị!

## Nguồn tải ảnh miễn phí:

- **Pexels**: https://www.pexels.com/vi-vn/
- **Unsplash**: https://unsplash.com/
- **Pixabay**: https://pixabay.com/vi/

## Lưu ý:

- Kích thước khuyến nghị: 600x400px hoặc 800x600px
- Format: JPG hoặc PNG
- Dung lượng: < 500KB/ảnh để tối ưu tốc độ
